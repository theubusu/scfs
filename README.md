# SCFS
Reverse-engineering Sony's SCFS filesystem.   
All work in progress

## Repository  
`/bin` - Dumped SCFS binaries (offline)  
`/images` - Dumped and created SCFS images (offline)   
`/misc` - Misc. notes and scripts   
`/notes` - Main notes and documentation  
`/scfs_rs` - Rust implementation of SCFS   
`/scfs-gpl-src` - Sony GPL source code regarding SCFS
